O que é Git?

O Git é um sistema de controle de versão criado por Linus Torvalds, o mesmo criador do Linux. Ele permite que desenvolvedores acompanhem mudanças no código ao longo do tempo, colaborem em equipe e revertam alterações se necessário.

Com o Git, você pode:

  • Controlar versões de um projeto localmente.
  • Comparar e mesclar mudanças entre arquivos.
  • Trabalhar em várias ramificações (branches).
  • Manter um histórico detalhado de todas as alterações.

O que é GitHub?

O GitHub é uma plataforma online que usa o Git para armazenar e compartilhar códigos. Ele é muito usado por desenvolvedores para hospedar repositórios e colaborar em projetos.

No GitHub, você pode:

  • Armazenar seus projetos na nuvem.
  • Contribuir com projetos open source.
  • Criar portfólios de programação.
  • Publicar sites com o GitHub Pages.

Principais comandos do Git

  • git init – Inicia um novo repositório Git.
  • git status – Mostra o estado atual dos arquivos.
  • git add . – Adiciona alterações para serem salvas.
  • git commit -m "mensagem" – Salva as alterações no histórico.
  • git branch – Lista ou cria ramificações do projeto.
  • git checkout nome-da-branch – Muda de branch.
  • git merge nome-da-branch – Mescla uma branch com a atual.
  • git remote add origin URL – Conecta o projeto ao GitHub.
  • git push -u origin main – Envia os arquivos para o repositório remoto.
